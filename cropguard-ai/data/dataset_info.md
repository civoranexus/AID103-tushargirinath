## Dataset Name
PlantVillage Crop Disease Dataset

## Dataset Source
Publicly available agricultural image dataset containing labeled images
of healthy and diseased crop leaves.

## Crops Covered
- Tomato
- Potato
- Pepper
- Maize
- Apple

## Disease Classes (Initial Scope)
- Healthy
- Leaf Blight
- Leaf Spot
- Powdery Mildew
- Rust

## Data Type
- RGB leaf images
- JPG/PNG formats

## Purpose of Dataset
This dataset will be used to train a deep learning model to identify
crop diseases from leaf images and provide AI-driven advisory output
to farmers.

## Planned Usage
- Image preprocessing using OpenCV
- CNN-based disease classification
- Integration with Flask API for inference
